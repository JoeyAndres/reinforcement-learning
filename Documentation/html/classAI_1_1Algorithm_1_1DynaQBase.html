Detailed Description

template<class S, class A>
class AI::Algorithm::DynaQBase< S, A >

Abstract base class for DynaQ algorithms.

Template Parameters
SState data type.
AAction data type.

DynaQ algorithm is a QLearning algorithm that employs a model to "model" the environment. Thus, in every update, the algorithm can do \(n\) simulations, enabling faster convergence to "optimal" policy.
