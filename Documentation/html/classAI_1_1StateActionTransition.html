Detailed Description

template<class S>
class AI::StateActionTransition< S >

StateTransition

This module is represents the possible transition states. Every call to StateTransition::update(state, reward), will increase the value of its frequency and at the same time update the reward value. Call to getNextState would then would return a state on the basis of how likely it occurs. Call to getReward would return the reward of the given state.

Parameters
greedydetermines how random is getNextState() is. A value of 1.0 means getNextState returns based on current likelihood of a state occuring (not random). With 0.0, it will not rely on the likelihood of state and return a random nextstate.
stepSizedetermines how the frequency is updated. A low value yields to a more accurate model of the environment but slower in learning environment. A value of 1.0 yields to forgeting the frequency information of all other transition states, suitable for deterministic environment.
