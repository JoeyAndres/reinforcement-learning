\hyperlink{classAI_1_1Algorithm_1_1DynaQ}{Dyna\-Q} algorithm is a \hyperlink{classAI_1_1Algorithm_1_1QLearning}{Q\-Learning} algorithm that employs a {\itshape model} to \char`\"{}model\char`\"{} the environment.
